JavaScript wishlist Test

In this test, we're going to write a small application that lets us add products to a wishlist. Please note that it's 100% client side and we don't store any data, so if you refresh your page: the wishlist will be emptied.

This test has 7 products, each product has its Add to wishlist button. When we click on this button, the product will be added to the wishlist, which will then be refresh and displayed.
